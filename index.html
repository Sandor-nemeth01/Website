Who I am

I am an IT student who likes to program, as well as gain certifications.

I've dabbled in Java, aiming to make programs for both android and Windows. I've also done C++ which is what I'm currently focusing on. I also know how to make websites in html and CSS, although that's my least favorite thing to do. I've got a certification in networking fundementals from Microsoft, a certification in Powerpoint, another in Excel, and I'm currently working on one for Access.

Java

Java is an object oriented programming language which assembles it's code to byte code for execution, as opposed to something like C which compiles to machine code. It's also portable as java code can easily be compiled on any platform, such as Windows, Mac, and Linux.

Web Development

I have some experience with making websites, as I've done several online courses for it, as well it having been a focus of my junior year of schooling. It isn't my favorite thing to do however, as designing something takes me forever.

C++

C++ is an extension of the C programming language, modified to be an object oriented programming language. It's faster than Java due to how it compiles to machine code instead of bytecode, although it isn't necessarily as easy to use. However the amount of additional libaries makes it incredibly useful. I don't know too much C++, but I'm looking forward to learning more as time goes on.

Certifications

My biggest certification is the networking fundementals from Microsoft. It took the most time to study for and was the hardest, although definitely was worth the time invested. Excel and powerpoint were both pretty easy to get, and will most likely help with whatever career I choose to persue in the future.
